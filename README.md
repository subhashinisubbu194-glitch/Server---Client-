# Java Socket Programming – Client Application

## Description

This project demonstrates the client side of a simple chat application using Java Socket Programming. The client connects to a server running on `localhost` at port `5000` and sends multiple text messages to the server.

## Features

* Connects to a server using Java Sockets
* Sends messages to the server
* Uses `PrintWriter` for output communication
* Demonstrates basic client-server interaction
* Simple and easy-to-understand networking implementation

## Technologies Used

* Java
* Socket Programming
* TCP Communication
* PrintWriter

## How It Works

1. The client creates a socket connection to the server on port 5000.
2. After a successful connection, it sends text messages to the server.
3. The server receives and displays the messages.
4. The client closes the connection after sending all messages.
